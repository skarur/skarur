# Breaking an object

1. In the **Class Pane**, select the parent class of the merged object.Ensure that the object that you want to break is the current object.
2. Drag the architect to one of the corners of the object, but not at the first helper.
3. In the command line, run **=b \[space]aBrokenObject\[Enter]**.

Note: 'aBrokenObject' stands for the object name. You can name the object as per your preference.
